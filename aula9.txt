Stored Procedures
	Blocos de código sql armazenados no BD
Vantagens
	Centralização
	Segurança
	Performance/Velocidade
	Suporte a transações

Limpeza de registros vencidos
	Storedprocedures pode verificar a data do registro
e limpar de acordo com um intervalo de tempo setado.

Sintaxe

Criando
	CREATE STORE PROCEDURE Nome
Utilizando
	CALL Nome
	EXECUTE Nome
Excluindo
	DROP PROCEDURE Nome

TRIGGERS

Vantagens
	Centralização
	Segurança
	Performance/Velocidade
	Suporte a transações
	+
	Execução de código SQL baseado em evento.
Tipos
	BEFORE INSERT
	BEFORE UPDATE
	BEFORE DELETE

Sintaxe
	CREATE TRIGGER Nome Tipo ON Tabela
	DROP TRIGGER Nome
